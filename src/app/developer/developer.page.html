About the Developer

Hi, I’m Sidhant Pandey, the creator of this app. I built this platform to combine fitness with technology, making it easy for everyone to track and improve their wellness journey.

This app started as a small project but grew into something powerful through consistent learning, teamwork, and dedication.

Technologies Used The tools that powered this app Ionic + Angular for UI Node.js for backend services Firebase for authentication & database Capacitor & REST APIs for integrations Git & GitHub for version control Journey of Development

The app was crafted step by step — from designing a simple wireframe to building real-time chat, fitness tracking, and a seamless user interface. I focused on optimizing performance and ensuring the app feels lightweight and fast.

Building this app taught me a lot about modern frameworks, teamwork, and creating something impactful for real users.
